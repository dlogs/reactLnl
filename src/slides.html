What is React

React is a declarative, component-based javascript library for composing dynamic web pages.

What makes React special?

  • Just the View
  • Virtual DOM
  • JSX Syntax
  • One-way data flow
  • Plain Javascript
  • Server-side rendering

React Components

  • Building blocks
  • use props and state, implement render()
  • Built in, Custom built, or 3rd Party

var MyComponent = React.createClass({
    render() {
        return (
            
Hello World

    }

ReactDOM.render(<MyComponent />, document.getElementById('example'))

Props and State

Props - passed into component State - modified by component

JSX

  • HTML-like syntax to make composing components easier
  • Transpiled via Babel to javascript

// JSX
return <MyComponent foo='Hello' bar={world} ><div>React!</div></MyComponent>
                

// transpiled Javascript
return React.createElement(
    MyComponent, {
        foo: 'Hello',
        bar: world
    }, React.createElement('div', null, 'React!')
);

Controlled Inputs


var MyComponent = React.createClass({
    onValueChanged(e) {
        this.setState({
            value: e.target.value
        });
    }
    render: function() {
        return (
            <div>
                <input type='text'
                       value={this.state.value}
                       onChange={this.onValueChanged} />
            </div>
        );
    }
});

Uncontrolled Inputs


var MyComponent = React.createClass({
    render: function() {
        return (
            <div>
                <input type='text'
                       defaultValue={this.props.defaultValue} />
            </div>
        );
    }
});
